Campyuk

The Campyuk project aims to create camping service providers with various facilities that users can enjoy. The Campyuk project has three user role models, namely Guest, Host, and Superadmin. Main feature in the guest role is to booking campsite from host. Main feature in the host role is add campsite to rent by guest. Main feature of the superadmin role is rejecting or accepting camping spots that were previously registered by the host.

Happy Shop

The Happy Shop project is an online shop (e-commerce) system to sell only the best items online. Registration, login, landing pages where users can view a list of items on the website; adding items to the cart; adding products to the sale list; the order page; transaction and shipping; sale and purchase history; and a user profile page are all features of this project.
